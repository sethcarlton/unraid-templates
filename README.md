# Unraid Templates

Unofficial Unraid Docker templates for self-hosted applications.

This repository maintains Unraid template metadata. Application behavior, setup details, and troubleshooting belong upstream unless they are specific to the Unraid template.

## Templates

| App | Template | Project | Docs |
| --- | --- | --- | --- |
| Executor | `templates/executor.xml` | https://github.com/RhysSullivan/executor | https://executor.sh |

See [NOTICE.md](NOTICE.md) for third-party project and licensing notes.

## Support

Use this repo's issues for template-specific problems, such as incorrect Unraid paths, ports, variables, icons, or Community Applications metadata.

Report application bugs upstream to the project listed above.
